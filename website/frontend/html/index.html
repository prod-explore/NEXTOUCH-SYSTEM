FAQ

Common Questions

Does it work without internet?

Yes. Nextouch works entirely over your local WiFi network. No internet needed for daily use.

How low is the latency?

Under 16ms over WiFi — comparable to Bluetooth mice. USB tethering provides even lower latency.

Is my data safe?

Absolutely. No accounts, no cloud, no telemetry. Everything stays between your phone and PC.

Why do I need two apps?

The desktop app receives input and controls your mouse. The mobile app captures your touch gestures.

Mac or Linux support?

Currently Windows only. Mac and Linux versions are planned based on user interest.

Can I use it on multiple PCs?

Yes. Install the desktop server on each PC. Pro licenses can be activated on up to 3 machines.
